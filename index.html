03

核心项目

PROJECT 01 · 第 27 届中国机器人及人工智能大赛 · 机器人应用赛(智能驾驶方向)

无人驾驶竞速车智能导航

角色 · 队长(带领 4 人团队,负责方案规划、技术路线决策与最终调试)

作为队长,带领团队基于 ROS2 + Nav2 从零搭建无人竞速车自主导航系统,打通感知—决策—控制全链路,参加全国级机器人比赛。

核心贡献
  • 实现 SLAM 建图与全局 / 局部规划,完成 30–50 m 复杂赛道(折弯、环形、锥桶)的端到端自主导航。
  • 自研 YOLO + OpenCV 视觉模块,实现红绿灯与人行道精准识别,配合 3 秒强制停车规则引擎显著降低违规罚时。
  • 完成激光雷达 / IMU / 编码器多传感器融合 + Nav2 参数调优,系统稳定性在队内测试中明显提升。
成果国赛二等奖 · 打通感知到控制的工程闭环,夯实智能驾驶全栈能力。
PROJECT 02 · 与中核集团合作 · 广东省攀登计划资助 · 挑战杯学术赛道省赛

天眼地足 · 核电站地空协同智慧巡检

角色 · 核心成员 / 轮足机器人运动控制 & 强化学习负责人

面向核电站高辐射、狭窄空间等极端环境,开发轮足机器人 + 背载无人机协同巡检系统,主导轮足平台的运动控制与强化学习落地。

核心贡献
  • 实现轮足复合运动:2 m/s 轮式高速移动 + 足式越障爬楼(25 cm 障碍、60° 楼梯)的动态智能切换。
  • 基于 PPO 训练轮足"运动小脑",真机上完成高辐射、狭窄空间中的稳定行走、动态平衡与姿态恢复。
  • 搭建分层决策架构:上层 HD-SLAM 感知 + 全局规划,下层强化学习负责局部鲁棒执行,并打通与背载无人机的协同接口。
成果项目获中核合作立项 + 攀登计划资助 + 挑战杯省赛入围;负责的轮足 RL 运控模块完成实机交付。
PROJECT 03 · 全球人形机器人自由格斗联赛 · 25 自由度全身强化学习控制

EngineAI T800 人形机器人全身运控

角色 · 核心成员 / 强化学习运动控制负责人

负责 EngineAI T800(25 DOF)人形机器人全身运动控制——从 Isaac Lab 仿真训练贯通至真机部署,为格斗联赛提供运动底座。

核心贡献
  • 基于 PPO + iFSQ 在 Isaac Lab 训练全身策略,覆盖行走、平衡、姿态切换与摔倒恢复。
  • 经多轮 Sim-to-Real 迭代(领域随机化、参数标定),将 ONNX 策略部署至 T800 真机,完成行走、格斗动作过渡与摔倒自主起身。
  • 参与 8 卡分布式训练 + 175+ 动作语料库建设,提升策略在动态扰动下的泛化能力。
成果策略作为团队运动控制底座,支撑高强度对抗场景下的实机稳定运行。
